# Alinhamento e ordenação de blocos
Os alinhamentos e e ordenação de blocos presentes no Bootstrap estão possuem garantias de funcionamento em áreas que fazem uso do [flexbox](https://developer.mozilla.org/pt-BR/docs/Web/CSS/CSS_Flexible_Box_Layout/Basic_Concepts_of_Flexbox). No Bootstrap, qualquer área pode ser transformada em "área flexbox", fazendo uso da classe `d-flex`. O sistema de grid apresentado anteriormente já faz uso do flexbox, portanto tudo o que será apresentado aqui funcionará com o sistema de grid.
